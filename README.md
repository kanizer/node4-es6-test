## Node4 / ES6 Testing

Test available ES6 features in Node 4.2.4 - uses Express preview server at :8080.

#### Setup

Install global npm tools:

```
npm i -g npm-check-updates
npm i -g node-inspector
```

#### Update npm modules

```
ncu
or
ncu -u
```

#### Start the dev server with

```
npm start
```

#### Start the debugger with

```
npm run debug
```
